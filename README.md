# Deploy Folder

Questa cartella contiene i file da caricare su GitHub per il deploy su Vercel.

## Istruzioni

1. Carica tutto il contenuto di questa cartella su GitHub.
2. Assicurati che i file di configurazione (package.json, vercel.json, ecc.) siano presenti.
3. Vercel rileverà automaticamente il progetto Next.js e procederà al deploy.
